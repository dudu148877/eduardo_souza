React

React utiliza uma arquitetura de componente, promovendo a modularidade e manutenção simplificada em aplicações. Sua capacidade de renderização no lado do servidor (Server-Side Rendering) e integração com outras bibliotecas e frameworks, tornam-no uma escolha popular para o desenvolvimento de aplicações web modernas e escaláveis

PHP

PHP é uma linguagem de script amplamente usada para desenvolvimento web, permitindo a criação de páginas dinâmicas e interativas em servidores web. Ela é especialmente adequada para a construção de aplicativos web e sites.

MySQL

O MySQL é um sistema de gerenciamento de banco de dados relacional amplamente utilizado, conhecido por sua confiabilidade e desempenho. Ele é usado para armazenar, recuperar e gerenciar dados em aplicativos e sites
